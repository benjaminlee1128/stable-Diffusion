# stable-diffusion-react-nextjs-mui-pwa

_PWA Web App front end for Stable Diffusion, on React/NextJS with Material UI_

Copyright (c) 2022 by Gadi Cohen <dragon@wastelands.net>. MIT Licensed.

## To Develop

1. Clone repo
1. `yarn install`
1. edit `.env.local`
1. `yarn dev`
